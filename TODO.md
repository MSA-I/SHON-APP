## 🔴 קריטי — חייב לפני שימוש

### 1. Build EXE עם הצבעים החדשים + auto-tag
- הקוד עודכן אבל EXE עדיין לא נבנה מחדש
- להריץ: `npm run tauri build` בתיקיית `app/`
- יצור: `app/src-tauri/target/release/bundle/nsis/Shon Blaish_0.1.0_x64-setup.exe`

### 2. תיוג ויזואלי — קטגוריות נוספות
- **מפות מפיות** ✅ — מופה ויזואלית (520 תמונות)
- **עיצובים שידרוג** ❌ — 160 תמונות, טרם תויגו ויזואלית
- **ריזורט בסיס** ❌ — 94 תמונות, טרם תויגו
- **חופות** (4 קטגוריות) ❌ — ~110 תמונות, טרם תויגו
- **אולם עיצוב בסיס 2026** ✅ — יש שמות עברית, auto-tag שם עובד כבר

### 3. מחיקת תיוג ישן + הרצת pass מחדש
- אחרי עדכון auto-tag, צריך לנקות את ה-IndexedDB ולהריץ שוב את ה-Tagging Pass
- כי אחרת הלייבלים הישנים (מהגרסה הקודמת) ישארו בלי הצבעים החדשים
- דרך: Settings → "אפס נתונים מקומיים" → פתח מחדש → pass ירוץ אוטומטית
